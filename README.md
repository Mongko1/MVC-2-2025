# Emergency Shelter Allocation System

## Getting started

```
https://github.com/Mongko1/MVC-2-2025.git
cd MVC-2-2025
```

## Prerequisite
Make sure you have:

- Docker

- Docker Compose

## Running with Docker Compose (Recommend for local development)
### 1. Build and start all services
```
docker compose up --build
```
this will start:
- Backend (FastAPI)
- Database (PostgreSQL)

### 2. Stop all services
```
docker compose down
```
